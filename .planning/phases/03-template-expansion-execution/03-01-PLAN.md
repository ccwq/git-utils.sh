**is_complex_shell_command(text: str) -> bool:**

**expand_env_vars(text: str) -> str:**
展开 %VAR% 环境变量（Windows 风格），每次执行时从 os.environ 读取。
使用正则 r'%([^%]+)%' 匹配并替换。

**expand_template(template: str, captures: list[str], rest_capture: str, runtime_args: list[str]) -> tuple[str, int]:**
Per D-19: 从后向前 scan 替换 $1, $2, $$
- 从后向前遍历 captures（len(captures) 到 1），替换 \$$ci
- 替换 \$\$ 为 rest_capture
- 处理 -- 占位符（D-20）：
  - 如果模板中有 --，runtime_args 插入到该位置
  - 如果没有 --，runtime_args 追加到末尾
- 返回 (final_command, exit_code_hint)

具体实现 per shell L1012:
```python
# 从后向前替换确保 $10 不会被误匹配
for ci in range(len(captures), 0, -1):
    final_template = final_template.replace(f'${ci}', captures[ci - 1])
final_template = final_template.replace('$$', rest_capture)
```

**invoke_cmd(cmd_text: str) -> int:**
执行命令并返回退出码。
Per D-22 和 D-23:
1. 调用 expand_env_vars 展开 %VAR%
2. 调用 is_complex_shell_command 检测复杂命令
3. 如果不是复杂命令：用 subprocess.run 执行，捕获退出码
4. 如果是复杂命令：用 subprocess.run(["bash", "-c", cmd_text]) 执行
5. 退出码映射：
   - 成功: 0
   - 一般错误: 1
   - 命令未找到 (FileNotFoundError): 127
