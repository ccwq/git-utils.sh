  <action>
修改 `parse_file` 函数支持目录 glob 加载和前缀语法：

1. 新增 `parse_dir(dir_path: str) -> Tuple[List, List]` 函数：
   - 使用 `glob.glob("*.txt")` 匹配 1 层深度
   - 过滤掉以下划线 `_` 开头的文件
   - 按文件名字母顺序处理
   - 合并所有文件的 aliases 和 errors

2. 修改 `parse_line` 函数支持前缀语法：
   - `&amp;alias_name` 前缀：`&amp;foo` 表示顺序执行模板（序列执行）
   - `|alias_name` 前缀：`|foo` 表示或执行模板（遇成功即停）
   - 前缀存储在 AliasEntry 或解析结果中，传递给 config.py 处理

示例：
```
&amp;foo echo %EDITOR%
&amp;foo echo %userprofile%
# 等价于 "echo %EDITOR% && echo %userprofile%"

|foo echo %EDITOR%
|foo echo %userprofile%
# 等价于 "echo %EDITOR% || echo %userprofile%"
```
</action>
